# 한빛 컨텐츠 폼
## 기능 명세
- 모든 내용은 options.json에 들어있습니다.
### TEXTAREA (NON-HTML)
- 순수 TEXTAREA
### TEXTAREA (NON-HTML-2)
- 순수 TEXTAREA이나, 자동으로 줄바꿈 br이 들어가는 방식
### TEXTAREA (HTML)
- TEXTAREA 내부에 HTML을 삽입하는 방식
### TEXTAREA (EDITOR)
- TEXTAREA지만, 스마트에디터가 들어가는 방식
### TEXT (NON-HTML)
- 한줄 텍스트 방식 (HTML 사용불가)
### TEXT (HTML)
- 한줄 텍스트 방식 (HTML 사용가능)
### NUMBER
- 숫자만 집어넣을 수 있는 방식
### IMAGES
- 이미지를 업로드하는 방식 (이미지만 업로드 가능)
